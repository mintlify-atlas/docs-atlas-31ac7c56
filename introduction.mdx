# Introduction to Origen

Origen is a creative overworld terrain generation config pack for Minecraft that transforms world generation with over 50 unique, handcrafted biomes featuring diverse and visually stunning terrain.

<Note>
Origen is currently in a testing phase, so bugs and issues are to be expected. Please report any issues you encounter on the [GitHub repository](https://github.com/Rearth/Origen).
</Note>

## What is Origen?

Origen is a configuration pack for [Terra](https://terra.polydev.org), a powerful Minecraft world generation plugin. While Terra provides the generation engine, Origen delivers the creative vision—reimagining Minecraft's overworld with dramatic landscapes, unique biome mechanics, and carefully crafted terrain features.

Built on top of Terra's default overworld config pack v2.0, Origen preserves familiar terrain features like trees, flora, and palettes while introducing completely new biome designs and generation techniques.

## What Makes Origen Unique?

### Creative Terrain Generation

Origen focuses on creating diverse and creative terrain without relying on custom content. Each biome tells a visual story:

- **Foliage Fortress**: Giant plateaus with sharp overhangs supported by massive stone pillars
- **Snowy Spires**: Terracotta pillars reaching 100 blocks high, partially frozen with ice chunks
- **Lush Loops**: Jungle environments with massive stone arches looping through the sky
- **Mesa Monuments**: Badlands filled with towering terracotta pillars
- **Inferno Isles**: Underground lava lakes with platform islands hanging on chains

### Intelligent Biome Distribution

Origen uses advanced distribution techniques to ensure biomes generate naturally:

- **Cellular noise** creates uniform biome sizes and shapes
- **Continental values** separate land and ocean cells cleanly
- **Coastal systems** with three categories: none, narrow, and wide
- **River variants** that flow through mountains using cave systems
- **Special biomes** like canyons that can carve through any terrain

### Comprehensive Environment Types

Origen includes biomes for every environment:

- **Land biomes**: Mountains, forests, deserts, tundras, and unique formations
- **Cave biomes**: Massive underground caverns with distinct themes
- **Coastal biomes**: Dramatic cliffs and transitions between land and sea
- **Ocean biomes**: Deep trenches, stone arches, and frozen seas
- **Special biomes**: Sinkholes and canyons that ignore normal distribution

## Terra Integration

Origen is built specifically for Terra and requires it to function. Terra is a modern world generation platform that:

- Replaces Minecraft's default world generation
- Supports highly customizable configuration packs
- Works with Paper, Fabric, and Forge
- Provides powerful tools for terrain generation

Origen leverages Terra's **3D noise generation** system and **biome pipeline v2** for sophisticated terrain control.
